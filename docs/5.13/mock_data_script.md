# 2026-05-13 Mock Data Script Memo

## 対象

- `uv run generate-mock-data --config ...`
- 実装本体は `src/scripts/generate_mock_data.py`

## 出力仕様

- 生成ごとに `data/raw/YYYYMMDDHHMMSS/` を作成する
- その配下に設定ファイルで指定した CSV 名を保存する
- 現在の想定ファイル名
  - `mock_acc_only.csv`
  - `mock_with_gyro.csv`
- 既存 CSV を毎回上書きしない運用にする

## CSV カラム

- 共通
  - `timestamp`
  - `elapsed_ms`
  - `acc_x`
  - `acc_y`
  - `acc_z`
  - `label`
- ジャイロ ON のときだけ追加
  - `gyro_x`
  - `gyro_y`
  - `gyro_z`

## ラベル定義

- `0`: 平坦
- `1`: 段差
- `2`: 粗さ

## 時系列生成ルール

- サンプル間隔は初期値 `0.25s` で `4Hz`
- ラベルは `0 -> 1 -> 2` の固定順ではなくランダム遷移
- ただしサンプル単位ランダムではなく，区間単位で継続時間を持たせる
- 初期の継続時間レンジ
  - `0=平坦`: `5s - 20s`
  - `1=段差`: `1s - 3s`
  - `2=粗さ`: `3s - 10s`
- 初期のラベル重み
  - `0`: `0.55`
  - `1`: `0.15`
  - `2`: `0.30`

## 波形の考え方

- `平坦`: 低振幅で緩やかな揺れ
- `段差`: 短時間の大きいスパイク + 反動
- `粗さ`: 細かい高周波揺れが連続
- ジャイロは完全独立ではなく，加速度変化にある程度相関させる

## 設定ファイル

- `configs/mock_data_acc_only.yaml`
- `configs/mock_data_with_gyro.yaml`
